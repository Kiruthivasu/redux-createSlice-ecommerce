# 🛒 Redux Toolkit E-Commerce App

A modern eCommerce web application built using React and Redux Toolkit (createSlice) for efficient state management.

## 🚀 Features
- Product listing page
- Add to cart functionality
- Remove from cart
- Increase / Decrease quantity
- Cart total calculation
- Redux Toolkit state management
- Clean and simple UI

## 🧠 Tech Stack
- React JS
- Redux Toolkit
- React Redux
- JavaScript (ES6+)
- CSS


## ⚙️ Setup Instructions

### Install dependencies
npm install

### Run project
npm start

## 🧩 Redux Concepts Used
- createSlice()
- configureStore()
- useSelector()
- useDispatch()

## 📌 Learning Outcome
- Redux Toolkit workflow
- Cart state management
- Component-based React structure
